# 3D-FPSLearning 
Class project made for the purpose of learning about the basics of 3D and FPS games using Unity.

## Video
https://youtu.be/v9th81CtNWc

## Features
- Basic FPS movement and camera
- Shooting and reloading
- Taking damage to Health and Sheild points
- Shooting gallery with moving and static targets
- Items for recovering Ammo, Health and Shield
- FSM enemy
  - Idle
  - Patrol
  - Alert
  - Chase
  - Attack
  - Hit
  - Die
- Lifebar for both enemies and player
- Doors / Keys
- Simple checkpoints
- Bullet decals
- Simple gun and enemy drone animations

## What I Learned
- Using Unity's **3D Character Controller**
- Using Unity's **NavMesh** component for enemy pathfinding
- Using Unity's **Terrain** tool
- Implement basic movement based on the player's current rotation at all times
- Implement basic FPS camera controller
- And much more
